# 🧹 Agricultural Data Cleaning

## About This Project
Real agricultural datasets are messy. This project demonstrates handling missing values, removing duplicates and renaming columns using Python Pandas.

## Tools Used
- Python | Pandas | Google Colab

## Key Skills
- Handling missing values with fillna()
- Removing duplicates with drop_duplicates()
- Renaming columns professionally
